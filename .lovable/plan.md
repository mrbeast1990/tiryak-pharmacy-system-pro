

# تحويل تصميم بطاقة المنتج من أفقي إلى رأسي

## الملف: `src/components/order-builder/CompactProductItem.tsx`

### التغيير:
تحويل التخطيط من `flex items-center justify-between` (أفقي) إلى تخطيط رأسي بسطرين:

**السطر الأول**: اسم المنتج بالكامل بدون قص (`line-clamp-2` بدل `line-clamp-1`)، يأخذ العرض الكامل.

**السطر الثاني**: صف أفقي يحتوي على:
- يمين (RTL): السعر + تاريخ الصلاحية
- يسار (RTL): أزرار +/- والكمية والإجمالي الفرعي

```text
┌──────────────────────────────────────┐
│ ALFA VITAMIN E 1000IU 30CAP          │  ← سطر 1: اسم كامل
│──────────────────────────────────────│
│  [−] 0 [+]  52.90 د.ل    47.95 د.ل  │  ← سطر 2: تحكم + سعر
│             الإجمالي       السعر      │
└──────────────────────────────────────┘
```

### التفاصيل:
- الحاوية الرئيسية: `flex flex-col px-3 py-2.5`
- اسم المنتج: `text-sm font-semibold line-clamp-2` بدون `min-w-0` أو تقييد عرض
- السطر الثاني: `flex items-center justify-between mt-1.5`
  - جهة اليمين (RTL): السعر والصلاحية
  - جهة اليسار (RTL): أزرار الكمية + الإجمالي الفرعي
